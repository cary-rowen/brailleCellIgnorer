# Braille Cell Ignorer

一个用于忽略损坏或故障点显器单元的 NVDA 插件。

**注： 在本插件及 NVDA 相关点显器上下文中，“盲文单元”，即盲文方（Braille cell）——一个盲符所占的长方形位置。**

## 使用场景

- 您的点显器有一个或多个单元的针脚卡住或点阵损坏
- 您希望继续使用点显器而避免被异常的输出所干扰
- 您需要剩余的单元自动移位填补空缺

## 功能特性

- **忽略指定单元**：配置哪些单元应被视为不存在
- **内容自动移位**：盲文内容会无缝绕过被忽略的单元
- **定位键处理**：按下被忽略单元的定位键会被静默忽略
- **按设备保存配置**：设置会按点显器型号和单元数分别保存

## 设置选项

路径：**NVDA 菜单 → 选项 → 设置 → Braille Cell Ignorer**

| 选项 | 说明 |
|------|------|
| **点显器配置** | 选择要配置的点显器（当前连接的或历史记录） |
| **忽略的盲文单元** | 输入单元编号（从 1 开始，逗号分隔，如 `1, 5, 40`） |
| **移除配置** | 删除不再使用的历史点显器的设置 |

## 注意事项

- 单元编号从 1 开始（第一个单元 = 1）。
- 目前仅支持单行点显器
- 保存后更改立即生效
